## GitHub Actionsとは

GitHubが提供するCI/CDプラットフォーム。ビルド、テスト、デプロイメントのパイプラインを自動化することができる。

GitHub Actionsは下記の項目で構成され、これらを設定することで、Githubでの操作をトリガーとした処理を定義することができる。

- ワークフロー
- イベント
- ジョブ
- Actions
- ランナー
